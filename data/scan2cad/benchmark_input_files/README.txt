# Scan2CAD Benchmark Input Files

Go to for proper description: http://kaldir.vc.in.tum.de/scan2cad_benchmark/documentation

## This folder contains:

- 'cad_appearances_hidden_testset.json': This file contains information on the number of appearances of a CAD model appears per scene
- 'scannet_hidden_testset.csv': This file contains a list of all scenes that appear in the hidden test set which are used in this benchmark
